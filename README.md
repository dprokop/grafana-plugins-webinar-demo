# Grafana plugins demo panel

Demo panel presented during [Grafana Plugins webinar](https://www.youtube.com/watch?v=4RLoHg4G9MI) during [GrafanaCon 2020](https://grafana.com/about/events/grafanacon/2020/)


## Getting started
1. Install dependencies
```BASH
yarn install
```
2. Build plugin in development mode or run in watch mode
```BASH
yarn dev
```
or
```BASH
yarn watch
```
